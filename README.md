# j-fire
Clinical Research Financial Management Web App

This is a web app that streamlines clinical trial financial management for small teams.

⚙️ Usage

You can use J Fire to enter a budget based on a clinical trial protocol, and see financial data and insight right away. 

Once signed in, user can save a budget in a database.

User can then use the app to track a study's finances, reporting, receivables and payables.

All is encrypted end to end, and personal data is always confidential.
